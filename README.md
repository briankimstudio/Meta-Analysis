# Meta Analysis

Tutorial on how to conduct meta analysis in R

- Effect size. 

- SMD. Cohen's d. Hedges' g
  - **metacont()**

- Forest plot. Fixed effect model. Random effect model.
  - **forest()**

- Heterogeneity. Cochrane Q(p<0.1). Higgins I2
  - **metareg()**, **bubble()**
  
- Publication bias. 
  - Funnel plot. **funnel()**
  - Egger's test. **metabias()**
  - Begg's test. **metabias()**
